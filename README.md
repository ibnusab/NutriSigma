# 📖 About

**Nutri Sigma** is an Android-based nutrition application designed to help users understand and manage their daily nutritional requirements. The app provides BMI and BMR calculations, daily calorie recommendations, nutrition tracking, and healthy meal suggestions in a simple and modern interface.
